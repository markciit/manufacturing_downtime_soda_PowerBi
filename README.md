This Power BI dashboard provides an analytical view of manufacturing line efficiency and downtime. 
By correlating operator performance with product-related errors, the report identifies root causes for downtime and supports data-driven decisions for process optimization.

Project Goals
1. Assess Line Efficiency: Monitor overall productivity and individual operator performance metrics.
2. Identify Downtime Drivers: Pinpoint major sources of downtime across different product lines and processes.
3. Analyze Error Distribution: Distinguish between machine-related failures and operator-related issues.
4. Optimize Training: Provide actionable insights for targeted operator training and reliability improvements.

Tools & Techniques
1. Power BI: Designed interactive, high-density visuals for industrial KPIs.
2. Power Query: Performed complex data cleaning and transformation of manufacturing logs.
3. DAX (Data Analysis Expressions): Developed custom measures for efficiency ratios and error frequency.
4. Data Modeling: Implemented a robust data model with efficient table relationships for seamless filtering.

Key Features
1. Operator Leaderboard: Ranking and efficiency comparisons across shifts.
2. Downtime Breakdown: High-level and granular views of downtime by product and cause.
3. Root Cause Analysis: Visual distribution of machine vs. operator error types.
4. Reliability Matrix: Visibility into notable downtime factors to drive process reliability.
